## 概要

前端接入最常见的文件、url、纯文本等格式的空间数据，并进行可视化展示

## 格式列表

### 已支持格式

- 文件
-   geojson
- url
-   geojson
-   xyz
-   mvt(pbf)
- 文本
-   geojson

### 待支持格式

- 文件
-   shp
-   gltf
- url
-   3dtiles
-   gltf
-   terrain
- 文本
-   wkt
